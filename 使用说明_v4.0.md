# 🚀 星TAP 高速缩图 v4.0 使用说明

---

## 📸 小白入门指南（3步搞定！）

### 第一步：下载安装
- 从 GitHub 下载 `星TAP-高清缩图-Win-v4.0.0.zip`
- 解压到任意文件夹（比如桌面）
- 双击 `星TAP高速缩图4.0.exe` 即可运行

### 第二步：压缩图片
1. 直接把图片拖进程序窗口
2. 选择处理模式（微信优化/高清无损/自定义）
3. 点击"开始压缩"按钮

### 第三步：查看结果
- 处理好的图片会自动保存在原图片旁边
- 文件名会加上 `_custom` 后缀
- 双击"打开输出目录"就能看到！

---

## 🎯 三种处理模式，总有一款适合你

### 1. 微信优化模式（推荐！）
- **适合**：发朋友圈、微信群、公众号
- **效果**：自动压到 900KB 左右，微信上传不会二次压缩
- **画质**：几乎看不出区别，朋友圈完美！

### 2. 高清无损模式
- **适合**：对画质要求极高的场景
- **效果**：保留更多细节，文件稍大一些
- **画质**：专业级，适合打印或存档

### 3. 自定义模式（高手专属）
- **适合**：想自己控制所有参数
- **可调参数**：
  - 最大尺寸（像素）
  - JPEG 质量（1-100）
  - 目标文件大小（KB）
  - 输出格式
  - 自定义导出目录

## 🖱️ 右键发送到（超方便！）

### 如何设置？
1. 按 `Win + R` 键，输入 `shell:sendto`，回车
2. 把 `星TAP高速缩图4.0.exe` 的快捷方式复制进去
3. 完成！

### 如何使用？
1. 在文件夹里选中一张或多张图片
2. 右键 → 发送到 → 星TAP高速缩图4.0
3. 程序自动启动并开始处理！

---

## 📁 自定义导出目录

不想把压缩后的图片放在原位置？没问题！

1. 切换到"自定义模式"
2. 点击"更改"按钮选择输出文件夹
3. 处理后的图片都会保存到你选的目录
4. 想恢复默认？点"重置"按钮就行

## 🤖 AI 调用方式（开发者看这里）

### JSON 模式（推荐）
```bash
星TAP高速缩图4.0.exe --json
```

输入 JSON 配置：
```json
{
  "version": "1.0",
  "files": ["C:/图片/test1.jpg", "C:/图片/test2.png"],
  "mode": "custom",
  "max_dim": 1920,
  "quality": 85,
  "output_dir": "C:/输出目录"
}
```

### CLI 参数模式
```bash
# 单文件处理
星TAP高速缩图4.0.exe --input "图片.jpg" --output-dir "输出目录" --max-dim 1920

# 批量处理
星TAP高速缩图4.0.exe --input "图片目录" --mode wechat
```

## 📊 常见问题解答

### Q: PNG 图片压缩后怎么没变小？
A: v4.0 已经优化了 PNG 压缩！现在会用最高压缩级别，效果好多了！

### Q: 处理后的图片找不到了？
A: 检查一下"自定义导出目录"是不是设置了其他地方，或者看原图片旁边有没有带 `_custom` 后缀的文件。

### Q: 可以同时处理多张图片吗？
A: 当然可以！拖进去多张，或者用右键发送到多选，都没问题！

### Q: 配置文件在哪里？
A: 在 `C:\Users\你的用户名\AppData\Roaming\rust_image_compressor\config.toml`，不会随便放在桌面了！

## 🔧 右键菜单添加工具

如果觉得"发送到"不够方便，可以用 `右键管理单程序.exe` 来添加右键菜单！

1. 运行 `F:\右键管理单程序.exe`
2. 按照提示添加 `星TAP高速缩图4.0.exe` 到右键菜单
3. 以后直接右键图片就能选"用星TAP压缩"了！

## 💡 小技巧

- **微信发图**：先用"微信优化模式"压缩，上传不会被二次压缩，画质最好！
- **批量处理**：把整个文件夹拖进去，一键搞定所有图片！
- **透明 PNG**：PNG 的透明背景会完美保留，不用担心变黑！
- **EXIF 信息**：JPEG 图片的拍摄信息（相机型号、GPS 等）都会保留！
